# Spooky Website

## A Halloween themed website 

A spooky halloween themed website, made with three js, react fibre and gsap libraries to give a nice illustration of the power of three js and gsap combined.

- Cool 3d Model

- Nice animation using gsap

- Cool loading screen

- Nice music halloween themed

- A secret jumpscare

## Requirements

- A decent version of node and npm.

## How to run locally

1. Git clone the repository.

```
git clone https://github.com/Turbash/spooky
cd spooky
```

2. Install the dependencies

```
npm install
```

3. Run the website locally

```
npm run dev
```
